# alexlauderbaugh.com

Personal site of Alex Lauderbaugh. Built with Next.js, MDX, and Tailwind. 
Source for [alexlauderbaugh.com](https://alexlauderbaugh.com).

Content lives in `/content/`. Each `.mdx` file is a log entry, project page, 
or static page. Adding new content is creating a file and committing it.

Built collaboratively with Claude Code. See `/projects/alexlauderbaugh-com` 
for the process notes.
